## Conventions

- Files are **never deleted**. When a file is fully superseded, its status changes to `Superseded` in this table and a notice is added at the top of the file pointing to its replacement.
- When only **some sections** become obsolete, a `> **Section superseded**` callout is added inline above the affected section. If more than two sections are superseded, a `## Changelog` table is appended at the end of the file.
